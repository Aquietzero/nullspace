在射影几何中，"完全四边形"是最基本的组合图形之一——它只需要四个点（任意三点不共线），却能从中自然生长出六条边、三条对角线，以及一组神奇的**调和点列**。调和分割是交叉比等于 $-1$ 的特殊情形，它连接了射影几何中看似不相关的概念：完全四边形、极点极线、无穷远点的中点，乃至射影坐标的建立。可以说，调和分割是射影几何的"关节"，理解了它，整个理论的骨架就清晰了。

# 完全四边形

## 定义

**完全四边形**（complete quadrilateral）是由平面上四条直线（任意三条不共点）确定的图形，共有：
- **6 个顶点**：四条直线两两相交，$\binom{4}{2}=6$ 个交点
- **4 条边**：即这四条直线本身
- **3 条对角线**：连接"对顶点"（不在同一条边上的两个顶点）的连线

<aside>
💡 注意区分：**完全四边形**（quadrilateral，四条线六点）和**完全四点形**（quadrangle，四个点六线）是对偶的概念。完全四点形由四个点（任意三点不共线）出发，产生六条边和三条对角点；完全四边形由四条线出发，产生六个顶点和三条对角线。它们互为对偶。
</aside>

## 对角三角形

完全四边形的三条对角线两两相交，形成一个三角形，称为**对角三角形**（diagonal triangle）。完全四边形与其对角三角形构成射影几何中的基本配置。

## 通俗理解

完全四边形就是一个"完整的"四边形——不仅有四条边，还把所有可能的对角线都画出来。普通四边形（如正方形）只有 $2$ 条对角线，而完全四边形有 $3$ 条，因为它把四条边的所有六个交点都包含进来了。

# 完全四点形与调和点列

## 关键定理：调和构造

**定理**：设 $A, B, C, D$ 是完全四点形的四个顶点（任意三点不共线），令：
$$P = AB \cap CD, \quad Q = AC \cap BD, \quad R = AD \cap BC$$

则这三点是对角三角形的三个顶点。在直线 $PQ$ 上，连线 $PR$ 与 $PQ$ 的交点 $S$（来自完全四点形的第四条对角线结构）满足：

$$\text{直线 } PR \text{ 与直线 } AQ \text{ 的交点 } S = PR \cap AQ$$

构成调和点列 $(P, Q; R, S)$（交叉比为 $-1$）……

实际上，调和分割最干净的陈述如下：

**定理（完全四边形与调和分割）**：设完全四边形有四条边 $a, b, c, d$（任意三边不共点）。取其中一条对角线 $l$，设 $l$ 与另外两条对角线分别交于点 $E$ 和 $F$，设 $l$ 与边 $a, b$（一对"对边"）分别交于 $A, B$，与边 $c, d$（另一对对边）分别交于 $C, D$。则：

$$(A, B; E, F) = -1$$

即 $E, F$ 是 $A, B$ 的调和共轭对。

<aside data-title="证明">

利用射影变换，不妨将完全四边形标准化。

设完全四边形的四个顶点（完全四点形的四点）为：
$$P_1 = [1:0:0], \quad P_2 = [0:1:0], \quad P_3 = [0:0:1], \quad P_4 = [1:1:1]$$

六条边及其交点（6个顶点）：
- $P_1P_2$（$z=0$），$P_1P_3$（$y=0$），$P_1P_4$（$y=z$）
- $P_2P_3$（$x=0$），$P_2P_4$（$x=z$），$P_3P_4$（$x=y$）

三条对角线（每条连接不在同一边上的两顶点）：

对角顶点（两条不相交边的交叉连线）：
- $P_1P_2 \cap P_3P_4 = [z=0] \cap [x=y]$：设 $[x:y:0]$ 且 $x=y$，即 $D_1 = [1:1:0]$
- $P_1P_3 \cap P_2P_4 = [y=0] \cap [x=z]$：即 $D_2 = [1:0:1]$
- $P_1P_4 \cap P_2P_3 = [y=z] \cap [x=0]$：即 $D_3 = [0:1:1]$

取对角线 $D_1D_2$，其方向向量 $D_1 \times D_2 = [1:1:0]\times[1:0:1] = [1\cdot1-0\cdot0:0\cdot1-1\cdot1:1\cdot0-1\cdot1] = [1:-1:-1]$，即直线方程 $x - y - z = 0$。

现在取边 $P_1P_2$（$z=0$）与此对角线的交点：在 $x-y-z=0$ 且 $z=0$ 上，$x=y$，得 $A = [1:1:0] = D_1$。

……这里直接就是对角点，说明选取需要调整。

换一种更直接的方式来证明调和性：

取四点 $A=[1:0:0], B=[-1:0:0]\sim[-1:0:0]$（即同一方向，不合适）……

改用经典的线段调和分割：设完全四边形四顶点为 $A, B, C, D$（任意三点不共线），令：

$$E = AB \cap CD, \quad F = AC \cap BD$$

这是对角三角形两顶点。取直线 $EF$，令 $G = EF \cap AD$，$H = EF \cap BC$。

**断言** $(E, F; G, H) = -1$。

利用射影变换，将 $A, B, E, F$ 标准化为 $A=[1:0], B=[0:1], E=[1:1], F=[1:-1]$（在 $\mathbb{P}^1$ 上）。

此时 $C$ 是直线 $AE$ 和直线 $BF$ 的交点（……因为 $C$ 要满足 $AC \cap BD = F$ 的条件）。

直接用代数计算：在标准四点形 $A=[1:0:0], B=[0:1:0], C=[0:0:1], D=[1:1:1]$ 中：

$E = AB \cap CD$：直线 $AB$ 是 $z=0$（$z$ 轴上的点），直线 $CD$ 是 $[0:0:1]\times[1:1:1]=[0\cdot1-1\cdot1:1\cdot1-0\cdot1:0\cdot1-0\cdot1]=[-1:1:0]$，即 $-x+y=0$，即 $x=y$。交点在 $z=0$ 且 $x=y$：$E=[1:1:0]$。

$F = AC \cap BD$：直线 $AC$ 是 $[1:0:0]\times[0:0:1]=[0:0\cdot1-0\cdot0:0\cdot0-0\cdot1]=[0:-1\cdot1 - \ldots]$

直线 $AC$：$[1:0:0]\times[0:0:1] = [0\cdot1-0\cdot0, 0\cdot0-1\cdot1, 1\cdot0-0\cdot0] = [0:-1:0]$，即 $y=0$。
直线 $BD$：$[0:1:0]\times[1:1:1] = [1\cdot1-0\cdot1, 0\cdot1-0\cdot1, 0\cdot1-1\cdot1] = [1:0:-1]$，即 $x=z$。
$F = y=0 \cap x=z$：$[1:0:1]$。

$G = EF \cap AD$：
直线 $EF$：$[1:1:0]\times[1:0:1] = [1\cdot1-0\cdot0, 0\cdot1-1\cdot1, 1\cdot0-1\cdot1] = [1:-1:-1]$，方程 $x-y-z=0$。
直线 $AD$：$[1:0:0]\times[1:1:1] = [0\cdot1-0\cdot1, 0\cdot1-1\cdot1, 1\cdot1-0\cdot1] = [0:-1:1]$，方程 $-y+z=0$，即 $y=z$。
$G$：$x-y-z=0$ 且 $y=z$：$x=2y$，取 $y=1$：$G=[2:1:1]$。

$H = EF \cap BC$：
直线 $BC$：$[0:1:0]\times[0:0:1] = [1\cdot1-0\cdot0, 0\cdot0-0\cdot1, 0\cdot0-1\cdot0] = [1:0:0]$，方程 $x=0$。
$H$：$x-y-z=0$ 且 $x=0$：$y+z=0$，即 $H=[0:1:-1]$。

**验证 $(E, F; G, H) = -1$：**

四点都在直线 $l: x-y-z=0$ 上。取参数化：令 $x = y+z$，以 $[y:z]$ 为 $\mathbb{P}^1$ 坐标：
- $E = [1:1:0]$：$[y:z]=[1:0]$
- $F = [1:0:1]$：$[y:z]=[0:1]$
- $G = [2:1:1]$：$[y:z]=[1:1]$
- $H = [0:1:-1]$：$[y:z]=[1:-1]$

交叉比：
$$(E, F; G, H) = \frac{|EG|\cdot|FH|}{|FG|\cdot|EH|}$$

其中 $|PQ| = p_1 q_2 - p_2 q_1$（$[y:z]$ 坐标的行列式）：

$|EG| = 1\cdot1 - 0\cdot1 = 1$，$|FH| = 0\cdot(-1)-1\cdot1 = -1$，$|FG|=0\cdot1-1\cdot1=-1$，$|EH|=1\cdot(-1)-0\cdot1=-1$。

$$(E,F;G,H) = \frac{1\cdot(-1)}{(-1)\cdot(-1)} = \frac{-1}{1} = -1 \checkmark$$

调和关系得证。
</aside>

# 调和共轭的性质

## 唯一性

**定理**：给定直线上两点 $A, B$，对任意第三点 $C$（$C \neq A, B$），存在唯一的点 $D$ 使得 $(A, B; C, D) = -1$。

<aside data-title="证明">

设 $A, B, C$ 的坐标分别为 $a, b, c$（欧氏坐标，$a \neq b$，$c \neq a, b$）。由 $(A,B;C,D)=-1$：

$$\frac{(c-a)(d-b)}{(c-b)(d-a)} = -1$$

整理：$(c-a)(d-b) = -(c-b)(d-a)$

展开：$cd - cb - ad + ab = -cd + ca + bd - ab$

整理：$2cd - cb - ad - ca - bd + 2ab = 0$

$d(2c - a - b) = 2ab - ca - cb = c(a+b) - 2ab + 2ab - ca - cb$……

直接解：$d(2c-a-b) = (a+b)c - 2ab$，故 $d = \frac{(a+b)c-2ab}{2c-a-b}$（当 $2c \neq a+b$，即 $c$ 不是 $A, B$ 的中点时）。

若 $c = \frac{a+b}{2}$（$C$ 是 $A, B$ 的欧氏中点），则 $2c-a-b=0$，方程无有限解，即 $D$ 是无穷远点。这是调和共轭的特殊情形：**中点与无穷远点互为调和共轭**。

$d$ 的值由 $a, b, c$ 唯一确定，故 $D$ 唯一。
</aside>

## 特殊情形：中点与无穷远点

最美的调和关系之一：在欧氏直线上，点 $C$ 是线段 $AB$ 的**中点**，则 $C$ 与**无穷远点** $D = \infty$ 互为关于 $A, B$ 的调和共轭。

$$\left(A, B; \frac{A+B}{2}, \infty\right) = -1$$

这个结论将射影几何（调和分割）与仿射几何（中点）优美地联系起来：中点的概念在射影几何中的"影子"就是调和共轭！

## 调和共轭的几何构造

给定 $A, B, C$，用尺规作图（不需要量角器！）找到调和共轭 $D$：

1. 在直线 $AB$ 外任取一点 $P$
2. 连 $PA$、$PB$、$PC$，在 $PC$ 上任取一点 $Q$（不在直线 $AB$ 上）
3. 连 $QA$，交 $PB$ 于 $R$；连 $QB$，交 $PA$ 于 $S$
4. 连 $RS$，与直线 $AB$ 的交点即为 $D$

这个构造保证 $(A, B; C, D) = -1$，且结果不依赖于 $P$ 和 $Q$ 的选取——这正是射影不变性的体现。

<script>
;(function () {
  // Fixed points on the baseline
  const A = [1, 1], B = [8, 1], C = [3, 1]
  // Auxiliary point P above the line
  const P = [2.5, 5]
  // Q on line PC (parameterized)
  function lerp(u, v, t) { return [u[0]+t*(v[0]-u[0]), u[1]+t*(v[1]-u[1])] }
  const Q = lerp(P, C, 1.6)   // Q beyond C on PC extended

  // Line-line intersection helper
  function intersect(p1, p2, p3, p4) {
    const d1x = p2[0]-p1[0], d1y = p2[1]-p1[1]
    const d2x = p4[0]-p3[0], d2y = p4[1]-p3[1]
    const denom = d1x*d2y - d1y*d2x
    if (Math.abs(denom) < 1e-10) return null
    const t = ((p3[0]-p1[0])*d2y - (p3[1]-p1[1])*d2x) / denom
    return [p1[0]+t*d1x, p1[1]+t*d1y]
  }

  // R = QA ∩ PB
  const R = intersect(Q, A, P, B)
  // S = QB ∩ PA
  const S = intersect(Q, B, P, A)
  // D = RS ∩ AB  (harmonic conjugate)
  const D = intersect(R, S, A, B)

  const g = new Graph2D(document.getElementById('fig-harmonic-construction'), {

  // Baseline AB extended
  g.line(-0.3, 1, 9.3, 1, {stroke:'#888', lineWidth:1.5, dash:[4,3]})

  g.line(P[0],P[1], B[0],B[1], {stroke:'#aaa', lineWidth:1.2})

  // Q extended a bit below C

  // QA → R (blue)
  g.line(Q[0],Q[1], R[0],R[1], {stroke:'#2196f3', lineWidth:1.8})
  // QB → S (red)
  g.line(Q[0],Q[1], S[0],S[1], {stroke:'#e74c3c', lineWidth:1.8})
  // RS (green) — the key line giving D
  g.line(R[0],R[1], D[0],D[1], {stroke:'#27ae60', lineWidth:2})

  // Points on baseline
  g.point(A[0],A[1],{color:'#333',radius:6,label:'$A$',labelOffset:[-4,-16]})

  g.point(P[0],P[1],{color:'#555',radius:5,label:'$P$',labelOffset:[6,0]})

# 调和分割与极点极线

## 联系

调和分割与极点极线（在圆锥曲线的语境下）有深刻联系：

**定理**：设 $C$ 是圆锥曲线，$P$ 是平面上一点，$l_P$ 是 $P$ 关于 $C$ 的极线。过 $P$ 作任意一条直线 $m$，$m$ 与 $C$ 交于 $X, Y$，与极线 $l_P$ 交于 $Z$。则：

$$(X, Y; P, Z) = -1$$

即 $P$ 和 $Z$（极线与割线的交点）是圆锥曲线割线两端点的调和共轭。

<aside>
💡 这个定理可以理解为：极点 $P$ 的极线 $l_P$ 是所有关于 $P$ 对圆锥曲线的"调和极线"——过 $P$ 的每条割线，极线恰好截出 $P$ 的调和共轭点。
</aside>

# 完全四点形的射影意义

## 构造射影坐标系

完全四点形（四点任意三点不共线）在射影几何中扮演"坐标系"的角色：
- 取四点 $E_1=[1:0:0]$，$E_2=[0:1:0]$，$E_3=[0:0:1]$，$E_4=[1:1:1]$
- 这四点唯一确定一个射影坐标系（由"四点定唯一射影变换"）
- 完全四点形的对角三角形（$E_1E_2 \cap E_3E_4$ 等三点）正是坐标轴方向

调和分割则给出了在射影坐标系中"定位"中间点的方法，使得射影直线可以被"射影参数化"。

# 小结

- **完全四边形**：四条线（任意三条不共点），产生六顶点和三条对角线
- **完全四点形**（对偶）：四个点（任意三点不共线），产生六条边和三个对角点
- **调和分割**：完全四点形的一条对角线被另外两条对角线截出调和点列，交叉比等于 $-1$
- **调和共轭的唯一性**：给定 $A, B, C$，关于 $A, B$ 的调和共轭 $D$ 唯一确定
- **中点与无穷远**：欧氏中点与无穷远点互为调和共轭——调和分割联结了射影几何与仿射几何
- **与极点极线**：过极点的割线被极线调和截分——调和分割是圆锥曲线极理论的核心
- **坐标意义**：完全四点形确定射影坐标系，调和分割是射影直线参数化的基础
